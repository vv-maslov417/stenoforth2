Стенографический Forth на основе SPF-4.29:

1. Сокращение динамического инструментального контекста в кратковременной памяти программиста 
и тем самым освобождение этой кратковременной памяти для динамического контекста решаемых программой задач:

1.1 За счет использования статических локальных переменных разных типов вместо слов манипуляции параметрами на стеке.

2.2 За счет упрощения использования контекстных словарей путем переключения контекстов поиска вместо 
выстраивания иерархии контекстов на стеке контекстов.

2. Макросы как инструмент для минимизации текста программ с возможностью программирования 
сверху вниз за счет возможности практически неограниченной вложенности макросов друг в друга 
и замены одинаковых отрезков текста именами макросов. Попутно макросы обеспечивают инлайнинг на уровне инструкций процессора.

3. Создание кода для группы литеральных имен одним определением, состоящим из опознавателя имен этой группы
и генератора кода для имен этой группы. При появлении во входном потоке литерального имени из определенной группы таких имен
это имя будет опознано и будет сформирован ( и если требуется то и выполнен) соответсвующий код.  

4. Концепция формирования определений слов глобальной видимости с помощью создания локальных слов с областью видимости только внутри 
формируемых определений. Имена локальных слов различного типа в локальном словаре, привязанном к формируемому определению 
после окончания формирования определения уничтожаются. Это позволяет определять небольшое количество глобальных имен, не 
засоряя глобальный словарь FORTH, и упрощает именование локальных слов. 

4.1 Имеется возможность для связывания таких глобальных определений в группы, внутри которых есть общий доступ к локальным именам
глобальных слов таких групп. 
4.2 После включения в группу последнего такого определения словари глобальных определений группы затираются.

5. Возможность быстрого создания определений слов с односимвольным представлением входящих слов и их контекстов.

6. Типизирование статических локальных переменных и локальных слов для упрощения синтаксиса при их использовании.

7. Возможность писать определения с использованием встроенного форт-ассемблера с коротким представлением инструкций. 

 При этом все возможности SP-Forth сохраняются. 

Получить расширенную форт-систему(Стенографический Forth на основе SPF-4.29) можно следуя указаниям в файле Spf4290.f
